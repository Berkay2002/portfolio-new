\section*{Projects}

\cvevent{WikiLLM}{LLM-Maintained Knowledge Bases}{Personal}{%
TypeScript CLI and agent plugin that compiles raw project sources into cross-linked Obsidian markdown wikis for Claude Code and Codex. Features an npx vault wizard, generated AGENTS.md and CLAUDE.md schemas, ingest/query/lint skills, Marp output, and host-specific automation guidance. Built with TypeScript, Node.js, npm CLI, Obsidian Markdown, and agent plugin packaging.
}

\cvevent{FastTalk}{Real-Time Voice Stack}{Personal}{%
Real-time voice conversation system with STT$\rightarrow$LLM$\rightarrow$TTS pipeline and end-to-end WebSocket streaming. Features Kokoro and Chatterbox TTS engines, PydanticAI-driven LLM with tool calling, and AudioWorklet-based 48\,kHz audio I/O. Built with Python, FastAPI, vLLM, faster-whisper, Docker, CUDA, Next.js, and React.
}

\cvevent{Nexus}{Local Multi-Agent Harness}{Personal}{%
Open-source local agent platform that routes one prompt through a classifier, orchestrator, and research/code/creative subagents sharing a sandboxed filesystem. Features hot tools, discoverable MCP wrapper tools, provider-tier routing, Claude OAuth and Codex CLI model paths, and a Next.js execution UI. Built with TypeScript, LangGraph, DeepAgents, AIO Sandbox, Next.js, Docker, and MCP.
}

\cvevent{Oversee}{Ops \& AI Dashboard}{Personal}{%
Multi-tenant workshop management platform with role-based access and an AI assistant that parses Excel files and manages reports through tool-calling. Features human-in-the-loop approval dialogs and Swedish-first analytics dashboards. Built with Next.js 16, Supabase, Vercel AI SDK, TanStack Table, and Playwright.
}

\cvevent{Stats for Spotify}{Music Analytics Platform}{Personal}{%
Full-stack web app that tracks Spotify listening history, visualizes taste evolution, and lets users compare stats with friends. Includes in-browser playback via the Web Playback SDK, server-computed recap insights, and historical trend charts. Built with Next.js 16, Supabase, Spotify API, Recharts, and Row Level Security.
}

\cvevent{LiTHePlan}{University Course Planning}{Personal}{%
Course planning tool for Linköping University master's students. Browse 339 courses across 15 programs, build validated 90\,hp study plans with drag-and-drop, dual conflict detection, and a Cmd+K command palette. Built with Next.js 16, Supabase, Upstash Redis, and Playwright.
}

Website for a youth judo club with Google Calendar webhook sync, admin dashboard for event management, email notifications via Nodemailer, and an animated bento grid homepage. Built with Next.js 16, React, Google Calendar API, Redis, Clerk Auth, and Cloudinary.

\cvevent{Albyrådet}{Non-Profit Web Application}{Personal (Pro Bono)}{%
