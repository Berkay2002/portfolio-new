# Commands

## Setup
- Install deps with bun (preferred): `bun install`
- Alt: `npm install` if bun is unavailable

## Development
- Start dev server on :3000: `bun dev` (or `npm run dev`)

## Build
- Production build: `bun run build` (or `npm run build`)
- Preview production: `bun run start` (after build)

## Test
- No automated test suite yet; run lint to validate: `bun run lint`

## Other Commands
- Lint: `bun run lint` (ESLint via eslint-config-next)
- Add shadcn component: `bun x shadcn@latest add <component>`
